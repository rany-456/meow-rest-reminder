# 把你的猫咪素材放在这个文件夹

代码会自动按下面的**文件名**读取素材。把你的动图改成对应名字放进来即可，**不用改任何代码**：

| 文件名 | 用途 | 建议 |
| --- | --- | --- |
| `cat-walk.webp` | 猫咪走进来时播放的**行走循环**动图 | 侧面行走、能循环 |
| `cat-idle.webp` | 走到屏幕前后的**待机**动图 | 坐着摇尾巴 / 眨眼 |
| `cat-happy.webp` | 被**摸头**时的开心反馈动图 | 可以和 idle 用同一张 |

## 说明

- 三个文件**不必都准备**。只要 `cat-walk.webp` 存在，就会用真实素材；其余缺失项会回退成走路图。
- 如果**一个都没放**，插件会用内置的纯 CSS 占位猫，功能照常完整。
- **推荐格式：动图 WebP**（体积最小、Chrome 原生支持）。GIF / APNG / 普通 PNG/JPG 也能用，但需要在 `src/content.js` 顶部的 `CAT_ASSETS` 配置中同步修改对应文件名。
- 想换文件名或加更多状态（比如生气、睡觉），改 `src/content.js` 顶部的 `CAT_ASSETS` 配置即可。
