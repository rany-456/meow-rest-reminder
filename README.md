# 本喵命你去休息 🐱

> 一只住在浏览器里的猫咪，专门盯着你的工作时长——盯够了就来找你。

## 是什么

**本喵命你去休息** 是一个 Chrome 浏览器扩展。

当你在同一个网页上持续停留超过设定时间（默认 30 分钟），一只猫咪会从屏幕底部走出来，越走越近，站到你面前，提醒你该休息了。

不是冷冰冰的弹窗通知，而是一只猫——走到你屏幕前的那只猫。

## 功能

- **久坐计时** — 在同一页面持续停留达到设定时长后触发，切走/锁屏自动暂停（可关闭）
- **由远及近的出场动画** — 猫咪从屏幕底部走出来，一边走一边放大，真的走到你面前
- **摸头互动** — 点击猫咪，爱心飞出来，猫咪开心地抖动
- **两个选择** — 「好的，去休息」计时清零；「再忙 N 分钟」顺延后再来找你
- **设置面板** — 点扩展图标可调整触发时长、顺延时长、是否暂停计时
- **一键预览** — 不用等 30 分钟，点「预览」立刻看猫咪出场效果
- **无干扰** — 基于 Shadow DOM 实现，猫咪 UI 与网页样式完全隔离，互不影响

## 安装

本扩展尚未上架 Chrome 应用商店，通过开发者模式本地加载即可：

1. 下载或克隆本仓库到本地
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角打开 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」**，选择本项目根目录（含 `manifest.json` 的那层）
5. 打开任意普通网页，点扩展图标 → **「预览猫咪出场效果」** 立刻查看

> ⚠️ 计时只在 `http` / `https` 普通网页生效；`chrome://`、新标签页等内部页面无法注入。

## 放入你自己的猫咪素材

插件内置了一套默认猫咪素材，你也可以换成自己喜欢的：

把动图按以下文件名放入 [`assets/cat/`](assets/cat/) 目录，不需要改任何代码：

| 文件名 | 用途 |
| --- | --- |
| `cat-walk.webp` | 走进来的行走循环动图 |
| `cat-idle.webp` | 走到面前后的待机动图 |
| `cat-happy.webp` | 被摸头时的开心反馈动图 |

推荐使用动图 WebP 格式（体积小，Chrome 原生支持）。三个文件不必都准备，缺失的会自动回退。


## 技术说明

- **纯原生实现**：原生 JavaScript + HTML + CSS，无任何构建工具或第三方依赖，加载即用
- **Manifest V3**：使用 `chrome.alarms` 做持久心跳计时，避免 Service Worker 被回收导致 `setInterval` 失效
- **Shadow DOM 隔离**：猫咪 UI 与宿主网页样式双向隔离，既不被网页 CSS 污染，也不影响网页
- **计时口径**：只对「当前激活 + 窗口聚焦」的标签页累计；页面导航到新 URL 会重置计时
